# WeatherApp
Aplikacja do sprawdzenia aktualnej pogody. Korzysta z danych z portalu openweathermap.org 
Do odczytu danych użyto obiekty JSON.
Użyte technologie:
-Spring Boot
-Vaadin
-OkHttp
